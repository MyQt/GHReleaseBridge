Currently **GHReleaseBridge** only emits two errors.

|  ERROR CODE                       | MEANING                          |
|-----------------------------------|----------------------------------|
| GHReleaseBridge::NETWORK_ERROR    | Some network error.              |
| GHReleaseBridge::JSON_PARSE_ERROR | The give json input is not good. |
